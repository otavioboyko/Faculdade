Italico e Emphase

Os elementos são respectivamente representados pelas tags, segundo o W3C, o Italico, serve para indicar um termo tecnico, uma palavra de uma outra lingua, um pensamento, etc, já o emphase, ele define uma emphase, mas funcionalmente, os dois fazem as mesmas coisas, eles deixam o texto em italico
