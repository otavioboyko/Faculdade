O que e formatting?

Formatting, e um grupo de tags que possuem o trabalho de -como o proprio nome sugere- formatar um texto, dar mais detalhes a ele, dar um significado especial a ele. Abaixo, sera colocado todos as tags "formatting".

Bold É usada para enfatizar o texto, indicando que ele tem uma importância especial ou deve ser destacado visualmente como texto em negrito. Strong Italic Italic, usada para indicar ao navegador que o texto contido nelas deve ser exibido em itálico. Emphasized É usada para enfatizar o texto em itálico, indicando que o texto deve ser enfatizado semanticamente. Marked É usada para destacar parte do texto, geralmente com um fundo amarelo, indicando que ele é relevante ou de interesse especial. Smaller O small serve para definir um texto com tamanho menor em HTML Deleted O deleted serve para indicar que um texto deve ser mostrado como deletado em HTML Inserted O inserted serve para mostrar texto como inserido em HTML Subscript Superscript
