Seletores

Ola, nesse site você vera os seletores CSS

Use a lista de navegação abaixo para pular para o seletor desejado

O seletor id, requer o uso do # (jogo da velha ou hashtag) + nome do id, e é usado para localizar um elemento no CSS e JavaScript, so pode ser usado uma vez em uma pagina, pois foi criado para identificar um elemento exclusivo na pagina, e se usado fora disso, pode causar problemas em seu CSS ou JavaScript, apenas o primeiro id sera selecionado

O seletor class, requer o uso de . (ponto) + nome da class, e tem a mesma função e funcionalidade de um id, porem, podem ser usados mais de uma vez

O seletor por tag, ocorre quando você adiciona uma tag sozinha no CSS

O seletor universal, e usado para selecionar TODO o conteudo de uma pagina, utiliza-se o simbolo * (asterisco)

Seletores simples, são seletores normais

Um seletor combinado é usado para selecionar elementos com base em múltiplos critérios. Por exemplo, você pode selecionar elementos que tenham uma determinada classe e estejam dentro de um certo elemento pai ou que tenham uma determinada estrutura hierárquica, existem diversos seletores combinados, como os abaixo.

pseudo-classes

casam elementos baseadas em outras características que não são: seu nome, seus atributos ou seu conteúdo. Para ativalos, usa-se : ou ::.

A pseudo-classe :first-child, casa com o elemento que é o primeiro filho de um outro elemento.

:link e :visited, afetam o estado dos links visitados e não visitados.

:hover, afeta quando o mouse passa por cima do elemento mas não clica

:active, e aplicado clica em algo e permanece clicado
